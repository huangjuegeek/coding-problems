# 445. Add Two Numbers II
> https://leetcode.com/problems/add-two-numbers-ii

## Solution 1:
输入链表的最高位在头结点，需要使用栈来将每一位的值逆序，从最低位开始相加
```java
class Solution {
    public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode cur;
        Deque<Integer> stack1 = new LinkedList<>();
        Deque<Integer> stack2 = new LinkedList<>();
        
        for (cur = l1; cur != null; cur = cur.next) {
            stack1.push(cur.val);
        }
        
        for (cur = l2; cur != null; cur = cur.next) {
            stack2.push(cur.val);
        }
        
        int sum = 0, carry = 0;    // carry为进位值，sum为当前位的和
        ListNode head = null;
        while (true) {
            if (carry == 0 && stack1.isEmpty() && stack2.isEmpty()) {
                break;
            }
            
            if (!stack1.isEmpty()) {
                sum += stack1.pop();
            }
            
            if (!stack2.isEmpty()) {
                sum += stack2.pop();
            }
            
            sum += carry;
            carry = sum / 10;
            cur = new ListNode(sum % 10);
            sum = 0;
            
            // 由于结果链表的最高位在头结点，因此每一位相加的结果结点应插入链表的头部
            cur.next = head;
            head = cur;
        }
        
        return head;
    }
}
```
